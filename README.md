## 1. 项目概述

ShotGenie 是一个面向无人机智能拍摄的构图理解与自动调整系统。项目目标不是单纯让无人机“能飞”，而是让它能够理解用户拍摄意图、识别当前画面质量、结合构图规则与美学评分自动调整机位，并最终输出更符合用户目标的高质量画面。

项目整体围绕“用户意图理解 → 画面评分 → 构图分析 → 飞行调整 → 再评分”的闭环展开，目标是逐步实现从辅助建议到自动机位优化，再到智能化拍摄执行。

## 3. 系统目标

本项目希望解决以下问题：

1. 如何判断当前无人机拍摄画面“好不好”。
2. 如何把用户的自然语言拍摄需求转换成可执行的构图目标。
3. 如何根据评分与构图缺陷分析，生成可执行的飞行与云台调整建议。
4. 如何把美学评分、构图规则、场景深度与飞行控制串成可迭代优化的闭环。

## 4. 构图评价模块

这些模型负责根据拍摄画面给出评分，并可结合关键词指导飞行调整机位。

目前主要考虑三类方案。

### 4.1 基于 CLIP 的评分体系

#### 4.1.1 CLIP

- **原生能力**：将图像和文字映射到同一语义空间，可通过构图关键词，如 `rule of thirds`、`symmetry`、`cinematic framing` 等，计算语义相似度评分。
- **优势**：很适合接入 `LLM → 关键词 → CLIP评分 → 飞行调整` 这条链路。
- **实现思路**：可先使用 BLIP-2 等视觉问答模型识别构图要素，再由 CLIP 对目标关键词进行相似度评分。

#### 4.1.2 Aesthetic Score Fine-tuned CLIP

- 可采用社区版本的 **CLIP + MLP Aesthetic Predictor**，主要从色彩、画面平衡、平面构图等角度对图像进行评价。
- 可基于 **AVA Dataset**、**LAION-Aesthetics** 等美学数据集训练，直接输出 0 到 10 的美学分数。

### 4.2 专业摄影构图评分模型

#### 4.2.1 NIMA（Neural Image Assessment）

- 使用 Inception 或 ResNet 提取图像特征。
- 输出图像的美学评分分布，再进一步得到平均分。
- **优势**：能够直接给出比较稳定的美学评分，不依赖文字标签。
- **适用场景**：适合飞行过程中做实时画质与构图质量监测。

#### 4.2.2 AVA Dataset Fine-tuned CNNs

- 基于 AVA 数据集训练 CNN 模型。
- 可以同时支持风格分类（如人像、风景）与美学分数预测。

### 4.3 特定构图规则检测（暂不作为主方案）

这部分目前不作为核心方案，后期可作为辅助模块，或由 CV 侧独立实现。

#### 4.3.1 Rule-of-Thirds Heatmap Detector

- 检测主体是否位于三分线交点附近。
- 可用于即时调整无人机位置与云台角度。

#### 4.3.2 Symmetry & Leading Lines Detector

- 基于 OpenCV 与深度学习的几何分析模型。
- 用于检测画面中的对称结构和线条引导关系。

#### 4.3.3 Foreground–Background Depth Balance Model

- 使用深度估计模型，如 **MiDaS**、**Depth Anything**。
- 用于评估前景与背景的比例关系，辅助分析画面层次与空间感。

## 5. 构图拍摄模型

这一部分更关注“如何给出参考构图”和“如何自动建议机位”。

### 5.1 摄影师风格指导模型

#### 5.1.1 BLIP-2 + GPT-4V / LLaVA

- 输入实景图像。
- 模型通过自然语言分析当前构图缺陷。
- 再由大模型生成飞行或云台调整建议。
- 可与 CLIP 打分模块结合，形成“分析 → 打分 → 调整”的闭环。

#### 5.1.2 Photographer Prompt Models

- 使用 Stable Diffusion 或 ControlNet 生成符合目标风格的参考画面。
- 让无人机在拍摄时朝参考构图靠近。
